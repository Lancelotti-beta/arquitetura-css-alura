# arquitetura-css-alura

curso de arquitetura da Alura.

 ## Iniciando o projeto  

 - Organizar a estruturar os arquivos css do projeto;
 - Metodologia Atomic Design.
 
 - Conhecendo o BEM (Block Element Modifier)
 ```
    bloco
    bloco__elemento
    bloco--modificador
    bloco__elemento--modificador
 ```
